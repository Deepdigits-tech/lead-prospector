# Default ICP: Food & Beverage Manufacturing

This is the default ICP configuration that ships with lead-prospector. Users can override this by creating `.claude/lead-prospector.local.md` in their project.

## Qualification Criteria

### QUALIFIED — Food producers with batch production
- Bakeries, confectionaries
- Protein processing, meat processing
- Ready-to-eat meal assembly, frozen foods
- Snacks and packaged foods
- Co-packers, private label manufacturing
- Sauces/dressings/condiments IF packaged for retail/foodservice (bottles, pouches, jars)
- Bonus signals: high-mix, high-volume, packaging lines
- Company size: 50-5,000 employees (sweet spot: 100-1,000)

### NOT QUALIFIED
- **Bulk liquid**: oil production, bulk chemicals (NOT packaged sauces/dressings — those are OK)
- **Alcohol**: winery, brewery, distillery
- **Fish / seafood** processing
- **Kitchens / restaurants** / food service only
- **Farms / agriculture** only (no processing)
- **B2B ingredient suppliers**: selling bulk ingredients (whey powder, premixes) without packaging finished goods
- **Animal feed / pet food**
- **Pure distributor / wholesaler** (no manufacturing)
- **Software / consulting / services** / job boards
- Under 50 employees or over 5,000

### KEY JUDGMENT RULE
**Don't be too restrictive.** If a company packages products on a line (bottles, pouches, boxes, bags) — likely qualifies. The key question is: **do they have packaging/production lines?**
